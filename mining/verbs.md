# Common Japanese Verbs (Daily Use)

> 📌 **Format Key**:  
> - **Dictionary Form** = plain, citation form (e.g., 食べる)  
> - **-masu Stem** = used for polite form 食べ**ます**, and compounds (e.g., 食べ**にいく**)  
> - `✓` = transitive (takes `を`), `✗` = intransitive (takes `が` or `に`)  
> - Katakana forms are *loan-adapted verbs* (e.g., サボる = to slack off)

> 📌 **Critical Usage Notes for SVO→SOV Learners**:
> - 🔁 **Object-Verb Order**:  
>   In SOV, **object + particle + verb** is fixed:  
>   ✅ わたしは **コーヒーを のみます**。 *(I drink coffee.)*  
>   ❌ わたしは のみます コーヒーを。  
> - 🎯 **Transitivity Matters**:  
>   - **開ける (akeru)** = *I* open the door → **ドアを あけます**  
>   - **開く (aku)** = The door opens → **ドアが あきます**  
>   Mixing these is a very common learner error!  
> - 🧠 **する-verbs**:  
>   Most katakana nouns can become verbs with **～する** (e.g., **ググる** = to Google — from *Google* + *-ru*).  
> - ⏱️ **Time + Place + Object + Verb**:  
>   ✅ **あした は がっこう で ともだち と ごはん を たべます。**  
>   *(Tomorrow, at school, with friends, [I] eat lunch.)*  
>   → `[Time] は [Place] で [Companion] と [Object] を [Verb]`

### 🎯 Top 5 Essential Verb Patterns
| Japanese | Romaji | Structure Breakdown |
|----------|--------|---------------------|
| りんご**を** たべます。 | **ringo o tabemasu.** | [Object] + **を** + [Verb-masu] |
| えき**に** いきます。 | **eki ni ikimasu.** | [Destination] + **に** + [Intransitive Verb] |
| いぬ**が** います。 | **inu ga imasu.** | [Animate Subject] + **が** + **います** |
| ほん**が** あります。 | **hon ga arimasu.** | [Inanimate Subject] + **が** + **あります** |
| にほんご**が** わかります。 | **nihongo ga wakarimasu.** | [Topic] + **が** + **わかります** (not を!) |
