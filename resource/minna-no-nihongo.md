### summary :

kalimat positif
```
kata benda 1 は　kata benda 2 です
kata benda 1 も　kata benda 2 です　(juga)
```

kalimat negatif :

```
kb1 は kb2 ではありません　/ じゃありません
```

kalimat tanya :

```
kb1 は kb2 ですか
```

menggabungkan 2 kata :

```
トヨタ　の　しゃいん
```
