- mo (juga)

## date 

| Concept     | Kanji         | Romaji (Reading)           | Notes |
|-------------|---------------|----------------------------|-------|
| Day (of month) | 日            | **-nichi**                 | Used as a suffix after numbers (e.g., 9日 = kokonoka / kyū-nichi) |
| Date (full) | 日付          | **hiduke**                 | General word for "date" |
| Month       | 月            | **-gatsu**                 | Suffix after numbers (e.g., 1月 = ichi-gatsu) |
| Year        | 年            | **-nen**                   | Suffix after numbers (e.g., 2026年 = nisen-nijūroku-nen) |

### Years

Just say the number + 年 (ねん / nen)

2026年 → にせんにじゅうろくねん

(Literally: two-thousand twenty-six year)

### Months

- Number + 月 (がつ / gatsu)
- January = 1月 → いちがつ
- February = 2月 → にがつ
- March = 3月 → さんがつ
- April = 4月 → しがつ (note: not "よんがつ")
- May = 5月 → ごがtsu
- June = 6月 → ろくがつ
- July = 7月 → しちがつ (or rarely なながつ)
- August = 8月 → はちがつ
- September = 9月 → くがつ (not きゅうがつ)
- October = 10月 → じゅうがつ
- November = 11月 → じゅういちがつ
- December = 12月 → じゅうにがつ

### Dates

| Number | Day (日) – Kanji + Reading      | Month (月) – Kanji + Reading |
|--------|----------------------------------|------------------------------|
| 1      | 1日 – **tsuitachi**             | 1月 – **ichigatsu**          |
| 2      | 2日 – **futsuka**               | 2月 – **nigatsu**            |
| 3      | 3日 – **mikka**                 | 3月 – **sangatsu**           |
| 4      | 4日 – **yokka**                 | 4月 – **shigatsu**           |
| 5      | 5日 – **itsuka**                | 5月 – **gogatsu**            |
| 6      | 6日 – **muika**                 | 6月 – **rokugatsu**          |
| 7      | 7日 – **nanoka**                | 7月 – **shichigatsu**        |
| 8      | 8日 – **yōka**                  | 8月 – **hachigatsu**         |
| 9      | 9日 – **kokonoka**              | 9月 – **kugatsu**            |
| 10     | 10日 – **tōka**                 | 10月 – **jūgatsu**           |
| 11–31  | Use regular number + **-nichi** | (e.g., 11日 = jūichi-nichi) | Months only go up to 12 |

### Example: **January 9, 2026**

In Japanese, dates are written **year → month → day**.

| Format        | Japanese (Kanji/Kana)       | Romaji                          |
|---------------|-----------------------------|----------------------------------|
| Full date     | 2026年1月9日                | **nisen nijūroku nen ichigatsu kokonoka** |
| Alternative (with -nichi) | 2026年1月9日       | **nisen nijūroku nen ichigatsu kyū-nichi** *(less common for 9th)* |

✅ **Preferred**: Use **kokonoka** for the 9th day in natural speech/writing.

## kata sifat (keiyoushi)

ada dua adjective :

1. I-Adjective (Berakhiran -i),Atarashii (新しい),Baru ✨,Furui (古い),Lama/Tua 📜

2. Na-Adjective (Berakhiran -na*),Kirei (きれい),Bagus/Bersih/Cantik 🌸
,Yuumei (有名),Terkenal 🌟

```
[Benda] wa [Kata Sifat] desu.
```

Kono tokei wa atarashii desu. (Jam ini baru.)

Sono kaban wa kirei desu. (Tas itu bagus/bersih.)

### kata sifat negatif

akhiran i :

- 高い	Takai (Mahal) ->	高くない	Takakunai

- 安い	Yasui (Murah) ->	安くない	Yasukunai

- おいしい Oishii (Enak) -> おいしくない Oishikunai

Kono tokei wa takai desu. (Jam ini mahal.)

Kono tokei wa takakunai desu. (Jam ini tidak mahal.)

---

Jika akhiran na, add dewa arimasen :

Kono kuruma wa yuumei desu. (Mobil ini terkenal.)

Kono kuruma wa yuumei dewa arimasen. (Mobil ini tidak terkenal.)

Sono machi wa kirei dewa arimasen. (Kota itu tidak bersih/bagus.)

## Kata kerja 

kata kerja selalu di akhir kamlimat, dalam bentuk sopan biasanya berakhiran ~masu

- 食べます	Tabemasu	Makan

- 飲みます	Nomimasu	Minum

- 行きます	Ikimasu	Pergi

- 見ます MimasuMelihat/Menonton

```
[Subjek] wa [Objek] o [Kata Kerja].
```

contohnya : 

Watashi wa mizu o nomimasu. (Saya minum air.)

Kare wa terebi o mimasu. (Dia menonton TV.)

---

- Sekarang/Akan datang	~masu	Tabemasu	Makan / Akan makan

- Negatif (Tidak)	~masen	Tabemasen	Tidak makan

- Lampau (Sudah)	~mashita	Tabemashita	Sudah makan

- Lampau Negatif	~masen deshita	Tabemasen deshita	Tadi/Dulu tidak makan

### Negatif:

Watashi wa pan o tabemasen.

(Saya tidak makan roti.)

### Lampau:

Watashi wa koohii o nomimashita.

(Saya sudah minum kopi.)

### Lampau Negatif:

Kare wa terebi o mimasen deshita.

(Dia [tadi/kemarin] tidak menonton TV.)

### ~te (minta tolong atau minta melakukan sesuatu)

meminta tolong atau menyatakan sedang melakukan sesuatu.

- Tabemasu (Makan)	Tabete	Tabete kudasai (Tolong dimakan)

- Mimasu (Melihat)	Mite	Mite kudasai (Tolong dilihat)

- Nemasu (Tidur)	Nete	Nete kudasai (Tolong tidur/istirahat)

- Akemasu (Membuka)AketeAkete kudasai (Tolong buka [pintu])


| Akhiran Sebelum ~masu | Berubah Menjadi | Contoh (Masu → Te)      | Arti               |
|------------------------|------------------|--------------------------|--------------------|
| ~i, ~chi, ~ri         | ~tte (っ て)     | Kaimasu → Katte         | Membeli            |
|                        |                  | Machimasu → Matte       | Menunggu           |
|                        |                  | Kaeririmasu → Kaette    | Pulang             |
| ~mi, ~ni, ~bi         | ~nde (んで)      | Nomimasu → Nonde        | Minum              |
|                        |                  | Shinimasu → Shinde      | Mati               |
|                        |                  | Asobimasu → Asonde      | Bermain            |
| ~ki                   | ~ite (いて)      | Kikimasu → Kiite        | Mendengar/Tanya    |
| ~gi                   | ~ide (いで)      | Oyogimasu → Oyoide      | Berenang           |
| ~shi                  | ~shite (して)    | Hanashimasu → Hanashite | Berbicara          |

Pengecualian Penting: Kata kerja Ikimasu (Pergi) berakhiran ~ki, tapi perubahannya bukan iite, melainkan Itte (行って).

### ~te imasu (Sedang Melakukan)

| Bentuk Sopan (~masu) | Bentuk ~te       | Sedang... (~te imasu)     | Bahasa Indonesia     |
|----------------------|------------------|----------------------------|----------------------|
| Tabemasu             | Tabete           | Tabete imasu               | Sedang makan         |
| Nomimasu             | Nonde            | Nonde imasu                | Sedang minum         |
| Mimasu               | Mite             | Mite imasu                 | Sedang menonton      |
| Benkyou shimasu      | Benkyou shite    | Benkyou shite imasu        | Sedang belajar       |

Watashi wa gohan o tabete imasu. (Saya sedang makan nasi.)

Tanaka-san wa terebi o mite imasu. (Pak Tanaka sedang menonton TV.)

### ~tai desu (Menyatakan Keinginan)

cukup mengambil kata kerja bentuk ~masu, buang masu-nya, lalu ganti dengan tai desu.

| Bentuk Sopan | Buang ~masu | Bentuk Ingin (~tai) | Bahasa Indonesia |
|--------------|-------------|----------------------|------------------|
| Tabemasu     | Tabe        | Tabetai desu         | Ingin makan      |
| Nomimasu     | Nomi        | Nomitai desu         | Ingin minum      |
| Ikimasu      | Iki         | Ikitai desu          | Ingin pergi      |
| Kaimasu      | Kai         | Kaitai desu          | Ingin membeli    |

Watashi wa Nihon e ikitai desu. (Saya ingin pergi ke Jepang.)

Atarashii kuruma ga kaitai desu. (Ingin membeli mobil baru.)

### ~nai de kudasai (Bentuk Larangan)

Untuk mengatakan "Tolong jangan..." atau "Mohon jangan...", kita menggunakan pola ~nai de kudasai. Sebelum menggunakan pola ini, kamu harus mengenal Bentuk Nai (Bentuk negatif dasar).

| Kelompok | Aturan Perubahan                                      | Contoh (Masu → Nai)     |
|----------|--------------------------------------------------------|--------------------------|
| Grup 2   | Buang ~masu, tambah ~nai                              | Tabemasu → Tabenai       |
|          |                                                        | Mimasu → Minai           |
| Grup 1   | Ubah bunyi -i menjadi -a, tambah ~nai                 | Kakimasu → Kakanai       |
|          |                                                        | Nomimasu → Nomanai       |
|          | Khusus akhiran "i" berubah menjadi "wa"               | Aimasu → Awanai          |
| Grup 3   | Hapalan (Irregular)                                   | Shimasu → Shinai         |

Tabenai + de kudasai = Tabenai de kudasai (Mohon jangan makan)

Nomanai + de kudasai = Nomanai de kudasai (Mohon jangan minum)

Ikanai + de kudasai = Ikanai de kudasai (Mohon jangan pergi)

Watashi no sushi o tabenai de kudasai.	私の寿司を食べないでください。	Tolong jangan makan sushi saya.

Ashita wa kaisha ni ikanai de kudasai.	明日は会社に行かないでください。	Besok tolong jangan pergi ke kantor.

Wasurenai de kudasai.	忘れないでください。	Tolong jangan lupa.

Hanasanai de kudasai.	話さないでください。	Tolong jangan bicara.

### ~ta koto ga arimasu (Menyatakan Pengalaman)

sekarang kita akan belajar cara mengatakan "Pernah melakukan sesuatu".

| Bentuk ~TE (N5) | Bentuk ~TA (N4) | Arti     |
|------------------|------------------|----------|
| Tabete           | Tabeta           | Makan    |
| Nonde            | Nonda            | Minum    |
| Itte             | Itta             | Pergi    |
| Kaite            | Kaita            | Menulis  |

```
[Bentuk ~TA] + koto ga arimasu
```

Tabeta koto ga arimasu. (Pernah makan.) 🍣

Itta koto ga arimasu. (Pernah pergi.) ✈️

### ~hou ga ii desu (memberikan saran, sebaiknya ...)

```
Saran Positif (Sebaiknya melakukan): [Kata Kerja Bentuk ~TA] + hou ga ii desu.
```

```
Saran Negatif (Sebaiknya tidak melakukan): [Kata Kerja Bentuk ~NAI] + hou ga ii desu.
```

行ったほうがいいです	Itta hou ga ii desu	Sebaiknya pergi

食べないほうがいいです	Tabenai hou ga ii desu	Sebaiknya jangan makan

薬を飲む	Kusuri o nomu	Minum obat

Sake o nomanai hou ga ii desu. (Sebaiknya jangan minum sake.)

Kusuri o nonda hou ga ii desu. (Sebaiknya minum obat.)

### bisa melakukan sesuatu (CAN/BISA)

| Grup | Aturan Perubahan                                      | Contoh (Bentuk Sopan → Bentuk Potensial)        | Arti (→ Arti Potensial)              |
|------|--------------------------------------------------------|--------------------------------------------------|---------------------------------------|
| Grup 1 | Ubah bunyi **-i** sebelum ~masu menjadi **-e**       | Ikimasu → Ikemasu                                | Pergi → Bisa pergi                   |
|      |                                                        | Nomimasu → Nomemasu                              | Minum → Bisa minum                   |
|      |                                                        | Kaimasu → Kaemasu                                | Membeli → Bisa membeli               |
| Grup 2 | Buang ~masu, ganti dengan **~raremasu**               | Tabemasu → Taberaremasu                          | Makan → Bisa makan                   |
|      |                                                        | Mimasu → Miraremasu                              | Melihat → Bisa melihat               |
| Grup 3 (Hafalan) | Bentuk tidak beraturan (irregular)          | Shimasu → Dekimasu                               | Melakukan → Bisa melakukan           |
|      |                                                        | Kimasu → Koraremasu                              | Datang → Bisa datang                 |

**Catatan Partikel**: Saat menggunakan bentuk potensial, partikel **o (を)** biasanya berubah menjadi **ga (が)**.  

Contoh: *Sushi ga taberaremasu.* (Bisa makan sushi.)

Byouki desu kara, neta hou ga ii desu. (Karena sakit, sebaiknya tidur.)

Karai tabemono o tabenai hou ga ii desu. (Sebaiknya jangan makan makanan pedas.)

Watashi wa Nihongo ga hanasemasu. (Saya bisa bicara bahasa Jepang.)

### ~tara (jika/ kalau)

Rumus: Gunakan Bentuk ~TA (yang sudah kamu pelajari) lalu tambah ~ra.

| Jenis             | Perubahan                          | Contoh                     | Arti             |
|-------------------|------------------------------------|----------------------------|------------------|
| Kata Kerja        | Bentuk ~TA + ra                   | Attara (Au ➡️ Atta + ra)  | Jika bertemu     |
| Kata Sifat -i     | Buang -i, + kattara               | Samukattara                | Jika dingin      |
| Kata Benda        | + dattara                         | Ame dattara                | Jika hujan       |

Nihon e ittara, sushi o tabetai desu. (Kalau pergi ke Jepang, saya ingin makan sushi.)

Samukattara, mado o shimete kudasai. (Kalau dingin, tolong tutup jendelanya.)
